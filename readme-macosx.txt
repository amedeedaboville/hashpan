RainbowCrack 1.2m MacOSX Readme

The source code will compile on MacOS X computers with PowerPC or Intel processors.  To compile, select the proper makefile (either .mac_ppc or .mac_intel) for your platform and run the command:

make -f makefile.mac_ppc

or

make -f makefile.mac_ppc64

or

make -f makefile.mac_intel

makefile.mac_ppc (G3, G4 Mac)
makefile.mac_ppc64 (G5 only)
makefile.mac_intel (Intel Mac only)
